## 1.clone项目（选一个存放文件夹打开cmd）
```
git clone https://github.com/AFanSKyQs/Apkanalysis.git
```
## 2.进入项目路径（编辑器如IDEA可忽略）
```
cd Apkanalysis
```

## 3.安装项目依赖

```
pnpm install
```

## 4.启动：用于开发环境的热重载和编译
说人话就是修改后保存完页面马上会自动刷新，不需要重启之类的，方便开发时的界面调节等~
```
pnpm run serve
```
---
### 生产环境的编译和压缩
```
pnpm run build
```

### 生产环境的编译和压缩并查看包分析器报告
```
pnpm run lint
```
