## 1.clone项目（选一个存放文件夹打开cmd）
```
git clone https://github.com/AFanSKyQs/Apkanalysis.git
```
## 2.安装项目依赖

```
pnpm install
```

## 3.用于开发环境的热重载和编译
```
pnpm run serve
```
---
### 生产环境的编译和压缩
```
pnpm run build
```

### 生产环境的编译和压缩并查看包分析器报告
```
pnpm run lint
```
